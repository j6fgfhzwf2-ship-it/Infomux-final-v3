# Infomux Final AI v3

Electron project with AI and interactive animations.

## Installation

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/Infomux-final-ai-v3.git
Copy .env.example to .env and put your API key:
cp .env.example .env
Install dependencies:
npm install
Launch the app:
npm start
Run as a standalone app
You can build a double-clickable version with:
npm run package:mac   # macOS
npm run package:win   # Windows
npm run package:linux # Linux
